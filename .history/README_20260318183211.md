# 🚗 Automatic Number Plate Recognition (ANPR)

An end-to-end Automatic Number Plate Recognition (ANPR) system built using **Python, YOLOv8, and EasyOCR**.
This project detects vehicles, extracts number plates, and reads plate text from images.

---

## 📌 Features

* 🔍 Detect vehicles using YOLOv8
* 📦 Extract number plate region
* 🔡 Recognize text using EasyOCR
* 🧠 Modular pipeline design (Detection → OCR → Processing)
* 🖼️ Works on images (extendable to video/webcam)

---

## 🛠️ Tech Stack

* Python
* YOLOv8 (Ultralytics)
* EasyOCR
* OpenCV




---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone <your-repo-link>
cd anpr-project
```

### 2️⃣ Create virtual environment

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🤖 Model Setup

This project uses a YOLOv8 model.

* Default: `yolov8n.pt` (auto-downloaded on first run)
* For better accuracy, use a **custom trained license plate model**

Place model inside:

```
models/
```

---

## ▶️ Run the Project

1. Add an image in:

```
data/raw/car.jpg
```

2. Run:

```bash
python main.py
```
